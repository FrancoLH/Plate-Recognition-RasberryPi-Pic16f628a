# Plate-Recognition-RasberryPi-Pic16f628a
Embedded AI-based vehicle license plate detection and access control system using YOLO, Raspberry Pi and PIC16F628A.

This is my final project for my Electronics Technician degree.

The project is divided into two main stages.

The first stage focuses on implementing YOLO on a Raspberry Pi for automatic license plate detection. Combined with OCR, the system is able to recognize license plate characters and compare them with a list of authorized vehicles.

The second stage focuses on implementing the PIC16F628A microcontroller to control the access barrier system. This includes managing the sensors and the barrier motor to ensure safe and proper vehicle access to the garage.
